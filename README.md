The site can be used to get Save My Exams pdfs for free. 

You type in the exact topic name (from the syllabus doc) and choose the options and then it either formats a google query or looks at its database if the pdf is already available.
It has dark and light modes, and a very fun UI to play with. 
The dots are randomly generated.
